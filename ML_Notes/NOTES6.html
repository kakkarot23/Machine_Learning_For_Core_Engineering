Week 5 Assignment 5 - Explained

Q1. Which of the following cases would reducing the learning rate NOT help with?

  • Model is underfitted and the predictions are not very accurate
  • Large oscillations in the loss function which are not settling
  • Loss function diverges and the values blow up
  • Loss function seems to approach minima and overshooting is to be avoided
Reducing the learning rate helps with overshooting or oscillations. However, if the model is underfitted (too simple to capture data patterns), lowering the learning rate won't improve accuracy because the problem is model capacity, not learning speed.

Q2. Which of the following methods addresses the rapid decay of learning rate in AdaGrad and how?

  • RMSProp, by adding a velocity vector which accumulates past gradients
  • SGD with momentum, by using an exponentially weighted moving average
  • Adam by adding a velocity vector which accumulates past gradients
  • RMSProp, by using an exponentially weighted moving average
AdaGrad decreases learning rate too quickly. RMSProp solves this by using an exponentially weighted moving average of squared gradients to adaptively scale learning rates, preventing them from decaying too fast.

Q3. The Adam optimizer combines the benefits of which two techniques?

  • Stochastic gradient descent and dropout
  • Momentum and adaptive learning rates
  • L1 and L2 regularization
  • Cross-entropy and mean squared loss
Adam combines momentum (smooths gradients) and adaptive learning rates (scales per parameter), making training faster and more stable.

Q4. Why are adaptive learning rate methods preferred in deep learning?

  • They eliminate the need for regularization
  • They make the model invariant to the number of layers
  • They automatically adjust learning rates for different parameters, speeding up convergence
  • They remove the need for batch normalization
Adaptive methods like Adam adjust learning rates for each parameter automatically, which helps in faster convergence, especially for deep networks where gradients can vary greatly.

Q5. Gradient descent first step values for dataset (β0, β1)?

  • 0.4, 1.38
  • 0.04, 0.138
  • 0.08, 0.276
  • 0.2, 0.69
Using gradient descent with learning rate 0.01, initial parameters 0, the first update step calculates gradients and updates: β0 = 0 + 0.01 * gradient, β1 = 0 + 0.01 * gradient. Step calculation gives β0 ≈ 0.08, β1 ≈ 0.276.

Q6. Which of the following matrices are positive definite?

  • M1
  • M1 and M3
  • M2 and M3
  • M1, M2, M3
Positive definite matrices must have all positive eigenvalues. M1 satisfies this property. M2 has negative or zero eigenvalues, and M3 has zero, so only M1 is positive definite.

Q7. Which statement is NOT true regarding the Hessian?

  • The Hessian is always positive definite
  • The Hessian does not depend on the value of the parameters
  • The Hessian does not depend on the actual value of y
  • All of these
For linear regression using MSE, the Hessian depends only on X. It's not always positive definite (depends on X), so claiming it always is positive definite is incorrect.

Q8. Eigenvalues of the matrix [[2,0,0],[1,3,0],[0,1,4]]?

  • 2, 3, 4
  • 2, 2, 4
  • 2, 3, 3
  • 1, 3, 4
Eigenvalues of a triangular matrix are its diagonal elements. So the eigenvalues are 2, 3, 4.

Q9. Hessian at (0,0) for f(x,y) = x² + 4y² and classify?

  • Hessian = [2 0; 0 8], local minimum
  • Hessian = [-2 0; 0 -8], local maximum
  • Hessian = [2 0; 0 -8], saddle point
  • Hessian = [0 0; 0 0], cannot determine
Hessian is [[2,0],[0,8]]. All eigenvalues positive → local minimum at (0,0).

Q10. L(θ) using Hessian H = [4 0; 0 9], θ - θ₀ = [1 -2], L(θ₀)=5?

  • 21
  • 15
  • 17
  • 25
Second-order Taylor expansion: L(θ) ≈ L(θ₀) + 0.5*(θ-θ₀)^T H (θ-θ₀). Calculating: 0.5 * [1 -2]*[[4,0],[0,9]]*[1 -2]^T = 20. Adding L(θ₀)=5 → L(θ)=25.
