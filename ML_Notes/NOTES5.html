Q1. Which of the following cases would reducing the learning rate NOT help with?

  • Model is underfitted and the predictions are not very accurate
  • Large oscillations in the loss function which are not settling

Correct Answer: Model is underfitted and the predictions are not very accurate

Q2. Which of the following methods addresses the rapid decay of learning rate in AdaGrad and how?

  • SGD with momentum, by using an exponentially weighted moving average
