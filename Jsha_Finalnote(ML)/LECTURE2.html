Topic: Broad Categorization of ML Algorithms

üìù One-Liner Notes

ML Algorithm Categories

  • **Supervised Learning** trains a model based on **labels** assigned to a **target variable**.
  • **Unsupervised Learning** does not require specified target variables or labels, seeking the data's **underlying structure**.
  • **Reinforcement Learning (RL)** involves a **virtual agent** receiving **rewards** based on favorable outcomes.
  • **Regression** is a type of supervised learning used to predict a **continuous variable** (e.g., temperature).
  • **Classification** is a type of supervised learning used to classify data into a **fixed number of categories** (e.g., overheated/not overheated).

Key ML Terms

  • A **Target variable** ($y$) is the variable of interest one desires to predict using a supervised ML model.
  • **Features** ($x$) are the independent variables that characterize each data point in the dataset.
  • **Parameters** ($\beta$) are the **learnable coefficients** (weights) adjusted during model training via optimization.
  • **Hyperparameters** ($\alpha$) are model configuration variables specified **before** training, determining model architecture.
  • A **Loss function** is the metric used to quantify the performance error between predicted and actual values.
  • **Cross validation** is a method to determine the **generalizability** of the model and choose hyperparameters rationally.

‚úÖ Multiple Choice Questions (MCQs)

**MCQ 1: Which type of Machine Learning is primarily used to discover the inherent patterns of data when labels are not specified?**

A. Regression B. Supervised Learning C. Unsupervised Learning D. Reinforcement Learning
Answer: C. **Unsupervised Learning** is characterized by the absence of specified target variables and the goal of learning the underlying structure of the data based on a similarity metric.

**MCQ 2: In a model predicting whether a chemical reactor has 'overheated' or 'not,' what type of supervised learning problem is this?**

A. Regression B. Binary Classification C. Clustering D. Dimensionality Reduction
Answer: B. This is a **Classification** problem because the output is one of a **fixed number of categories** ("overheated" or "not"). Specifically, with two categories, it is **Binary Classification**.

**MCQ 3: Which of the following defines the model configuration variables that are specified *before* the training process begins?**

A. Features ($x$) B. Target variable ($y$) C. Parameters ($\beta$) D. Hyperparameters ($\alpha$)
Answer: D. **Hyperparameters** are configuration variables specified **before training** that determine the model architecture and training process (e.g., learning rate, number of layers). **Parameters** are learned *during* training.

‚ùì Short Question & Answer

**Q1: Distinguish between the Training Set and the Test Set in ML.**

**A:**

  • **Training Set:** The collection of data points (features and corresponding target variable values) used by the algorithm to **train** the ML model, learning the relationships between inputs and outputs.
  • **Test Set:** A collection of **unseen data points** used to **independently verify** the performance and generalizability of the final model *after* training is complete.

**Q2: Give one example of a problem that requires Regression and one that requires Classification.**

**A:**

  • **Regression Example:** Predicting the **Voltage** (a continuous value) provided by a battery based on its type, age, and temperature.
  • **Classification Example:** Determining if a person is suffering from **chest congestion or not** (Yes/No) based on their chest X-ray and symptoms.

**Q3: What are Generative Algorithms and name two common examples.**

**A:** Generative algorithms are a broad type of unsupervised learning that **learn the distribution underlying the data**. This allows them to generate **new samples** which are statistically similar to the input training data. Examples include **Generative Adversarial Networks (GANs)** and **Variational Autoencoders (VAEs)**.

üìö Long Q&A / Description / Graphical Questions

**Q1: Explain the three broad categories of Machine Learning (ML) algorithms based on their training mechanism.**

**A: The three broad categories are Supervised, Unsupervised, and Reinforcement Learning:**

  • **Supervised Learning:**
    • **Mechanism:** Training happens based on specified values (labels) assigned to a target variable. It seeks to map each input feature vector to a known output value.
    • **Application:** Applies to the prediction of both continuous (Regression) and discrete (Classification) variables.
  • **Unsupervised Learning:**
    • **Mechanism:** Target variables are not specified, and labels are not required. The model learns the **underlying structure and inherent patterns** of the data based on some similarity metric.
    • **Application:** This is used for tasks like Clustering and Dimensionality Reduction (e.g., PCA).
  • **Reinforcement Learning (RL):**
    • **Mechanism:** A virtual agent takes **actions** in an **environment** and receives **rewards** based on favorable outcomes or **penalties** for unfavorable ones.
    • **Goal:** The maximization of the cumulative reward leads to an optimal policy or sequence of actions for solving the problem.

**Q2: Describe Supervised ML's role as a function approximator, defining its key components.**

**A:** Supervised Machine Learning acts as a complex function approximator that maps a set of input features to a target variable. This relationship is mathematically represented as: $$y=f(x_{1},x_{2},...,x_{n};\beta_{1},\beta_{2},...,\beta_{p};\alpha_{1},\alpha_{2},...,\alpha_{h})$$

Target Variable ($y$):
The variable of interest that the model is trained to predict. It can be a **continuous** value (for Regression) or a **discrete** category (for Classification).
Features ($x_{1}, \dots, x_{n}$):
The **independent variables** (inputs) that characterize each data point in the dataset. These are the observed values used to make the prediction.
Parameters ($\beta_{1}, \dots, \beta_{p}$):
The **learnable coefficients or weights** within the model. These values are iteratively adjusted by the algorithm during the **Training** phase using **multivariate optimization** to minimize the Loss Function.
Hyperparameters ($\alpha_{1}, \dots, \alpha_{h}$):
The **model configuration variables** that are set externally by the user *before* training starts. They determine the model architecture and training process (e.g., depth of a tree, number of layers in a neural network).
