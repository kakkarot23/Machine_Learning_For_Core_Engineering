Topic: Confidence Intervals and Hypothesis Testing for Linear Regression Parameters

üìù One-Liner Notes

Confidence Intervals (CI) and Significance Level ($\alpha$)

    [cite_start]
  • A **Confidence Interval (CI)** specifies an interval around a stochastic quantity where it will lie $c$ of times, where $c$ is the confidence level[cite: 1, 3].
    • [cite_start]
  • The **Significance Level ($\alpha$)** is a related quantity, where the confidence level $c = 1 - \alpha$[cite: 1, 3, 6].
    • [cite_start]
  • For a 95% confidence level, the significance level $\alpha$ is **0.05**[cite: 7].
    • [cite_start]
  • For a 99% confidence level, the significance level $\alpha$ is **0.01**[cite: 7].

Hypothesis Testing and Regression Parameters

    [cite_start]
  • **Hypothesis testing** validates a certain statement (the null hypothesis $H_0$) against an alternate hypothesis ($H_a$) at a given significance level[cite: 9, 14, 16].
    • [cite_start]
  • The **Z-test** is used for the population mean when the **population standard deviation ($\sigma$) is known**[cite: 19, 34].
    • [cite_start]
  • The **T-test (Student's t-test)** is used for the population mean when the **population standard deviation ($\sigma$) is unknown**[cite: 30, 67, 69].
    • [cite_start]
  • To assign a **Confidence Interval (CI)** to a linear regression parameter $\hat{\beta}$, the model errors are typically assumed to be **normally distributed**[cite: 78, 88, 89].

‚úÖ Multiple Choice Questions (MCQs)

**MCQ 1: If a linear regression model's confidence interval is 90%, what is the corresponding significance level ($\alpha$)?**

A. 0.90 B. 0.05 C. 0.10 D. 0.01
Answer: C. Confidence level $c = 1 - \alpha$. [cite_start]If $c=90\%$ (or 0.90), then $\alpha = 1 - 0.90 = 0.10$[cite: 7].

**MCQ 2: The Z-test is specifically used for the population mean under the condition that:**

A. The sample size $n$ is very small. B. The sample standard deviation ($s$) is known. C. The population standard deviation ($\sigma$) is known. D. The data is non-normally distributed.
Answer: C. The Z-test is applicable when the population standard deviation ($\sigma$) is known. [cite_start]If $\sigma$ is unknown, the t-test must be used[cite: 19, 34, 69].

**MCQ 3: For an unbiased estimator $\hat{\beta}$ of a parameter $\beta$ in linear regression, what is the required condition for its expected value?**

A. $E[\hat{\beta}] = 0$ B. $\text{Var}(\hat{\beta}) = \sigma^2$ C. $E[\hat{\beta}] = \beta$ D. $E[\hat{\beta}] = \hat{\beta}$
[cite_start]Answer: C. The expected value of an unbiased estimator $\hat{\beta}$ must be equal to the true parameter value $\beta$, as shown by the proof $E[\hat{\beta}] = \beta$[cite: 101, 104].

‚ùì Short Question & Answer

**Q1: Write the mathematical expression for the Z-statistic and state its distribution.**

**A:** The Z-statistic is defined as: $$Z = \frac{\overline{y}-\mu}{\sigma/\sqrt{n}}$$ [cite_start]The Z-statistic follows the **Standard Normal Distribution** ($\mathcal{N}(0, 1)$)[cite: 47, 53].

**Q2: Write the mathematical expression for the T-statistic and state its distribution and degrees of freedom.**

**A:** The T-statistic is defined as: $$T = \frac{\overline{y}-\mu}{s/\sqrt{n}}$$ [cite_start]The T-statistic follows the **Student's t-distribution** with **$(n-1)$ degrees of freedom** ($t_{n-1}$)[cite: 70, 72, 73].

**Q3: State the rejection rule for the Null Hypothesis ($H_0$) in a two-sided Z-test.**

[cite_start]

**A:** In a two-sided test, the Null Hypothesis ($H_0$) is rejected if the absolute value of the calculated Z-statistic is greater than the critical value $Z_{\alpha/2}$[cite: 55, 56]: $$H_0 \text{ is rejected if } |Z| > Z_{\alpha/2}$$

**Q1: Prove that the linear regression parameter estimator $\hat{\beta}$ is an unbiased estimator for the true parameter vector $\beta$.**

**A: Proof that $E[\hat{\beta}] = \beta$ (Unbiased Estimator):**

  1. **Parameter Estimator $\hat{\beta}$:** [cite_start]The optimal parameter vector is given by the Normal Equation $\hat{\beta} = (X^T X)^{-1} X^T Y$[cite: 110].
  2. **Actual Value $Y$:** The actual target vector $Y$ is modeled as the linear combination of features and true parameters ($\beta$) plus an error term ($\epsilon$): $$\mathbf{Y} = \mathbf{X}\mathbf{\beta} + \mathbf{\epsilon}$$ [cite_start]The expectation of the error is assumed to be zero: $E(\epsilon) = 0$[cite: 94, 100].
  3. **Expected Value of $\hat{\beta}$:** Substitute $\mathbf{Y}$ into the expression for $E[\hat{\beta}]$: $$E[\hat{\beta}] = E[(X^T X)^{-1} X^T (X\beta + \epsilon)]$$
  4. **Expansion and Simplification:** $$E[\hat{\beta}] = E[(X^T X)^{-1} X^T X\beta + (X^T X)^{-1} X^T \epsilon]$$ Since $(X^T X)^{-1} X^T X = I$ (Identity Matrix): $$E[\hat{\beta}] = E[I\beta] + (X^T X)^{-1} X^T E(\epsilon)$$
  5. **Final Result:** Since $E(\epsilon) = 0$, the second term vanishes: $$E[\hat{\beta}] = \beta + 0 = \beta$$ [cite_start]**Conclusion:** Since $E[\hat{\beta}] = \beta$, $\hat{\beta}$ is an **unbiased estimator** for the true parameter vector $\beta$[cite: 101, 104].

**Q2: Describe the formula for the Confidence Interval (CI) of a linear regression parameter $\beta$ and identify the components, assuming the error variance is estimated.**

**A: Confidence Interval for $\beta$:**

[cite_start]

To determine the Confidence Interval (CI) for a linear regression parameter $\beta$ (when the population error variance $\sigma^2$ is unknown and estimated by $\hat{\sigma}^2$ or $s^2$), the t-distribution is used[cite: 121, 122]. [cite_start]The formula for the variance of the parameter estimator is $Var(\hat{\beta}) = \sigma^2 \text{diag}((X^T X)^{-1})$[cite: 118, 119].

The CI for $\beta$ at a $1-\alpha$ confidence level is generally given by:

$$\hat{\beta} \pm t_{n-p, \alpha/2} \cdot \sqrt{\hat{\sigma}^2 \cdot \text{diag}((X^T X)^{-1})}$$

Where:

    [cite_start]
  • $\hat{\beta}$ is the **estimated parameter**[cite: 122].
    • [cite_start]
  • $t_{n-p, \alpha/2}$ is the **critical value** from the Student's t-distribution[cite: 122].
    • [cite_start]
  • $n$ is the number of data points and $p$ is the number of parameters, making $n-p$ the **degrees of freedom**[cite: 121].
    • [cite_start]
  • $\hat{\sigma}^2$ (or $s^2$) is the **estimated error variance**, calculated using the residuals[cite: 119].
    • [cite_start]
  • $\text{diag}((X^T X)^{-1})$ are the **diagonal elements** of the $(X^T X)^{-1}$ matrix, which relate to the variance of each parameter estimate[cite: 118].

**Q3: Describe the rejection region for the Null Hypothesis ($H_0$) in a One-Sided Test based on the provided graphical representation.**

**A: Rejection Region for One-Sided Test:**

[cite_start]

The provided graph illustrates a **one-sided hypothesis test** for a standard normal distribution, where the significance level $\alpha$ is concentrated in the right tail of the distribution[cite: 59, 60].

    [cite_start]
  • The critical value is denoted as $z_{1-\alpha}$ (or $z_{\alpha}$ for the $\alpha$ area from the right)[cite: 58, 65].
  • The shaded area $\alpha$ represents the **rejection region**.

  • The Null Hypothesis ($H_0$) is rejected if the calculated Z-statistic falls into this region[cite: 61, 62].

The formal rejection rule is: $$H_0 \text{ is rejected if } z > z_{1-\alpha}$$
