Topic: Conversion of Random Variables (Variable Transformations)

üìù One-Liner Notes

Variable Transformation Concepts

    [cite_start]
  • Programming languages often have built-in functions for sampling only a few specific probability distributions, such as the **Uniform or Normal distribution**[cite: 747, 748].
    • [cite_start]
  • **Variable transformations** are used to convert random variables sampled from a certain distribution to those sampled from another desired distribution[cite: 750, 751].
    • [cite_start]
  • The transformation from an initial random variable $X$ to a new variable $Y$ is defined by the function $Y = r(X)$[cite: 754].
    • [cite_start]
  • If the inverse function $r^{-1}(y)$ is an **increasing function**, the formula $G(y) = F[r^{-1}(y)]$ is used for the CDF[cite: 768, 769].
    • [cite_start]
  • If the inverse function $r^{-1}(y)$ is a **decreasing function**, the formula $G(y) = 1 - F[r^{-1}(y)]$ is used for the CDF[cite: 778, 779].
    • [cite_start]
  • The transformation $Y = \frac{1}{\alpha} \ln(\frac{1}{X})$ converts a Uniform distribution $X \sim U(0, 1)$ into an **Exponential distribution** $Y \sim Exp(\alpha)$[cite: 789, 815, 816].

‚úÖ Multiple Choice Questions (MCQs)

**MCQ 1: Which statement is true for the Cumulative Distribution Function (CDF), $G(y)$, when the inverse function $r^{-1}(y)$ is an increasing function?**

A. $G(y) = 1 - F[r^{-1}(y)]$ B. $G(y) = F[r^{-1}(y)]$ C. $g(y) = f(r^{-1}(y))$ D. $g(y) = -f(r^{-1}(y))$
[cite_start]Answer: B. For an increasing inverse function $r^{-1}(y)$, the CDF of $Y$ is simply the CDF of $X$ evaluated at the inverse function, $G(y) = F[r^{-1}(y)]$[cite: 769].

**MCQ 2: What does the negative sign signify in the PDF transformation formula $g(y) = -f(r^{-1}(y)) \cdot \frac{d(r^{-1}(y))}{dy}$?**

A. $r(x)$ is an increasing function B. $r(x)$ is not invertible C. $r^{-1}(y)$ is a decreasing function D. $X$ is a discrete random variable
[cite_start]Answer: C. The negative sign is required in the formula for $g(y)$ when the inverse function $r^{-1}(y)$ is a **decreasing function**[cite: 778, 797]. The term $\frac{d(r^{-1}(y))}{dy}$ will be negative, and the negative sign ensures $g(y)$ remains positive, as it must be for a PDF.

**MCQ 3: If a random variable $X$ is sampled from a Uniform distribution $U(0, 1)$, what is the corresponding distribution of the transformed variable $Y=\frac{1}{\alpha}ln(\frac{1}{X})$?**

A. Normal distribution B. Binomial distribution C. Exponential distribution $Exp(\alpha)$ D. Student's t distribution
[cite_start]Answer: C. The transformation $Y=\frac{1}{\alpha}ln(\frac{1}{X})$ applied to $X \sim U(0, 1)$ yields the Probability Density Function $g(y) = \alpha e^{-\alpha y}$ for $y \ge 0$, which is the PDF of an **Exponential distribution** $Y \sim Exp(\alpha)$[cite: 789, 815, 816].

‚ùì Short Question & Answer

**Q1: What is the necessity for Variable Transformations when dealing with probability distributions?**

[cite_start]

**A:** Variable transformation is necessary because programming languages typically only have built-in functions to sample from a few specific probability distributions (e.g., Uniform or Normal)[cite: 747, 748]. [cite_start]This technique allows variables sampled from these readily available distributions to be mathematically converted to those sampled from another desired, more complex distribution[cite: 750, 751].

**Q2: State the formula for the PDF of a transformed random variable $Y=r(X)$ when the inverse $r^{-1}(y)$ is an increasing function.**

**A:** The Probability Density Function (PDF) $g(y)$ of the transformed variable $Y$ is given by: $$g(y) = f(r^{-1}(y)) \cdot \frac{d(r^{-1}(y))}{dy}$$ [cite_start]where $f(x)$ is the PDF of $X$ and $r^{-1}(y)$ is the inverse transformation function[cite: 771].

**Q3: Give an example of a pair of inverse functions demonstrated in the lecture notes.**

**A:** An example of a pair of inverse functions is $r(x) = e^x$ and its inverse, $r^{-1}(x) = \ln(x)$. [cite_start]Applying the inverse function to $r(x)$ gives $r^{-1}(r(x)) = \ln(e^x) = x$[cite: 782].

**Q1: Explain the general formulas for calculating the PDF, $g(y)$, of a transformed random variable $Y = r(X)$, differentiating between increasing and decreasing inverse functions.**

**A:** For a random variable $X$ with PDF $f(x)$ transformed to $Y = r(X)$, the PDF of $Y$, denoted $g(y)$, is calculated based on whether the inverse function $r^{-1}(y)$ is increasing or decreasing.

[cite_start]
If the inverse function $r^{-1}(y)$ is an **increasing function**[cite: 768]:
The PDF is calculated without a negative sign:
[cite_start]
$$g(y) = f(r^{-1}(y)) \cdot \frac{d(r^{-1}(y))}{dy}$$ [cite: 771]
[cite_start]
If the inverse function $r^{-1}(y)$ is a **decreasing function**[cite: 778, 798]:
The PDF is calculated with a negative sign to ensure $g(y)$ remains positive, as $\frac{d(r^{-1}(y))}{dy}$ will be negative for a decreasing function:
[cite_start]
$$g(y) = -f(r^{-1}(y)) \cdot \frac{d(r^{-1}(y))}{dy}$$ [cite: 780, 797]

These two cases can be unified by using the absolute value of the derivative, which is often written as the Jacobian term: $g(y) = f(r^{-1}(y)) \cdot \left|\frac{d(r^{-1}(y))}{dy}\right|$

**Q2: Perform the transformation calculation for $X \sim U(0, 1)$ to $Y$ where $Y=\frac{1}{\alpha}ln(\frac{1}{X})$, $\alpha>0$, and identify the resulting distribution of $Y$.**

**A: Derivation of the distribution of $Y$:**

  1. **Given Distributions and Transformation:**
      [cite_start]
    • $X \sim U(0, 1)$, so the PDF of $X$ is $f(x)=1$ for $0 \le x \le 1$, and $0$ otherwise[cite: 801].
      • [cite_start]
    • The transformation is $Y = r(X) = \frac{1}{\alpha} \ln(\frac{1}{X})$[cite: 789].
  2. **Find the Inverse Function $r^{-1}(y) = x$:** $$y = \frac{1}{\alpha} \ln\left(\frac{1}{x}\right) \implies \alpha y = \ln\left(\frac{1}{x}\right) \implies e^{\alpha y} = \frac{1}{x} \implies x = e^{-\alpha y}$$ [cite_start]The inverse function is $r^{-1}(y) = e^{-\alpha y}$[cite: 796]. [cite_start]Since $\alpha>0$, this is a **decreasing function**[cite: 798].
  3. **Determine the range of $Y$:** Since $0 \le x \le 1$, we find the corresponding range for $y$ using the transformation:
    • When $x=1$, $y = \frac{1}{\alpha} \ln(\frac{1}{1}) = 0$.
    • When $x \to 0$, $y = \frac{1}{\alpha} \ln(\frac{1}{0}) \to \infty$.
    [cite_start]The range of $Y$ is $0 \le y < \infty$[cite: 813].
  4. **Calculate the Derivative:** $$\frac{d(r^{-1}(y))}{dy} = \frac{d}{dy}(e^{-\alpha y}) = -\alpha e^{-\alpha y}$$
  5. **Calculate the PDF $g(y)$ (using the decreasing function rule):** [cite_start]$$g(y) = -f(r^{-1}(y)) \cdot \frac{d(r^{-1}(y))}{dy}$$ [cite: 797] [cite_start]$$g(y) = -f(e^{-\alpha y}) \cdot (-\alpha e^{-\alpha y}) = \alpha e^{-\alpha y} \cdot f(e^{-\alpha y})$$ [cite: 801]
  6. **Substitute $f(x)$:** Since $f(x)=1$ for $0 \le x \le 1$, and $r^{-1}(y) = x = e^{-\alpha y}$ is within this range for $y \ge 0$, we have $f(e^{-\alpha y})=1$. $$g(y) = \alpha e^{-\alpha y} \cdot 1$$ [cite_start]$$g(y) = \begin{cases}\alpha e^{-\alpha y} & 0 \le y < \infty \\ 0 & \text{otherwise}\end{cases}$$ [cite: 808]

**Resulting Distribution:** This PDF, $g(y) = \alpha e^{-\alpha y}$, is the Probability Density Function of a **Exponential Distribution** with rate parameter $\alpha$. [cite_start]Thus, $Y \sim Exp(\alpha)$[cite: 815, 816].
